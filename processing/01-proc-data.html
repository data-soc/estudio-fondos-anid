3.1.3 Descripción Bases: postulaciones_limpia y postulaciones_limpia_sociales

La base de datos POSTULACIONES_2016_2025 contiene información sobre las postulaciones realizadas a los distintos fondos y programas de becas de la Agencia Nacional de Investigación y Desarrollo (ANID, ex CONICYT) entre los años 2016 y 2025. Esta base de datos abarca el universo de postulaciones institucionales, sobre la cual se aplicaron dos filtros fundamentales: en primer lugar, se seleccionó únicamente la disciplina “Ciencias Sociales” según la clasificación de la OCDE para producir una base de datos específica únicamente con los datos para las disciplinas de las ciencias sociales; y en segundo lugar, se conservaron exclusivamente aquellas postulaciones cuyo proceso de evaluación finalizó en condición de “Adjudicado” o “No Adjudicado”, excluyendo postulaciones inadmisibles o fuera de bases. El tamaño final de la muestra comprende 30.373 registros únicos en la base postulaciones_limpia y 8603 registros en la base postulaciones_limpia_sociales, lo que representa el universo de postulaciones a proyectos Fondecyt en el área de Ciencias Sociales durante el período analizado.

Entre las variables principales de esta base se encuentran identificadores únicos como CODIGO_PROYECTO, variables temporales como AGNO_FALLO que registra el año de resolución del concurso, y variables de clasificación institucional como SUBDIRECCION, PROGRAMA e INSTRUMENTO, que permiten categorizar el tipo de financiamiento (ej. Becas PFCHA, Proyectos FONDECYT, FONDEF, etc.). La variable ESTADO_RESOLUCION_CONCURSO indica el resultado final de la postulación. Además, la base incluye variables geográficas como REGION_EJECUCION y MACROZONA_MINCIENCIA, así como variables demográficas e institucionales del postulante, como SEXO. Otras variables como TIPO_BENEFICIARIO y TIPO_INSTITUCION fueron eliminadas posteriormente por contener información redundante, ya que en ambas variables el 99% de los casos correspondían a “PERSONA NATURAL” respectivamente, lo que no aportaba variabilidad significativa para el análisis.

El proceso de depuración aplicado a esta base incluyó la eliminación de columnas redundantes, la estandarización de cadenas de texto (eliminación de tildes, espacios en blanco y conversión a mayúsculas), y la transformación de variables textuales a formato de factor (factor) para optimizar el uso de memoria y facilitar los modelos estadísticos y análisis longitudinales posteriores.

                  DISCIPLINA_DETALLE, GRUPO_DE_EVALUACION, SEXO,

Considerando que tanto la base de postulaciones contiene también a los proyectos adjudicados, se realizó un cruce (left join) entre ambas bases utilizando CODIGO_PROYECTO como clave, con el objetivo de inyectar la información de adjudicaciones a cada postulación. Este proceso permitió enriquecer la base de postulaciones con variables adicionales provenientes de las adjudicaciones, como el nombre del proyecto, grupo de evaluación, disciplina detallada, duración, monto adjudicado, entre otros. De todas formas, la base de adjudicaciones cuenta con mayor granularidad que la base de postulaciones, por lo que hay ciertos casos que en tanto no estén adjudicados no cuentan con la información de adjudicación, lo que se refleja en valores NA en las variables provenientes de adjudicaciones para los proyectos no adjudicados. Sin embargo, esta estructura permite realizar análisis comparativos entre postulaciones y adjudicaciones, así como análisis longitudinales para comprender la evolución de los proyectos a lo largo del tiempo.

Al mismo tiempo, la variable de grupo de estudio no se encontraba bien equilibrada, por lo que se realizó un proceso de agrupamiento de las disciplinas detalladas en grupos de estudio más generales como detalla ANID para cada categoría, con el objetivo de facilitar el análisis posterior. Este proceso implicó revisar cada una de las disciplinas detalladas presentes en la base de adjudicaciones, y asignarlas a grupos de estudio más amplios que representaran áreas temáticas coherentes dentro de las ciencias sociales. Por ejemplo, disciplinas como “Antropología” y “Arqueología” que figuraban por separado, fueron agrupadas bajo el grupo de estudio “Antropología y Arqueología”. Este proceso se realizó para todas las disciplinas detalladas, resultando en un conjunto final de grupos de estudio que permiten analizar los proyectos adjudicados desde una perspectiva temática más generalizada, facilitando así la interpretación y comparación entre diferentes áreas dentro de las ciencias sociales.

3.1.6 Descripción de la base de adjudicaciones Fondecyt

La base de datos BDH_HISTORICA contiene el registro histórico completo de las adjudicaciones de fondos Fondecyt desde 1982 hasta 2025. Si bien el período temporal original de esta base abarca más de cuatro décadas de financiamiento científico, para efectos de este análisis se aplicó un filtro temporal que selecciona únicamente los años 2016 a 2025, con el propósito de mantener consistencia con la base de postulaciones. El universo de análisis corresponde a todos los proyectos adjudicados en concursos Fondecyt, sobre los cuales se aplicaron dos filtros principales: selección de la disciplina “Ciencias Sociales” según la clasificación de la OCDE y restricción al período 2016-2025.

Esta fue fusionada con la base de postulaciones anteriormente expuesta para aunarlas y dejar una base de datos final que contiene tanto a los proyectos adjudicados como a los no adjudicados, con el objetivo de enriquecer la información disponible para cada postulación con variables provenientes de las adjudicaciones. Quedando con dos productos finales: una base de datos completa con todas las postulaciones y adjudicaciones fondecyt (base_final_completa), y una base de datos específica para Ciencias Sociales con todas las postulaciones y adjudicaciones en esta área (base_final_sociales).

Las variables principales incluyen identificadores de proyecto como CODIGO_PROYECTO, variables temporales como AGNO_CONCURSO y AGNO_FALLO —que permiten distinguir entre el año de convocatoria y el año de resolución—, y variables de financiamiento como DURACION_MESES, que registra la extensión temporal del proyecto. El volumen de los recursos asignados se captura en la variable MONTO_ADJUDICADO, expresado en miles de pesos chilenos. La clasificación institucional se registra mediante las variables SUBDIRECCION, PROGRAMA e INSTRUMENTO, mientras que la clasificación disciplinaria se detalla en AREA_OCDE y GRUPO_DE_EVALUACION. Esta última variable es particularmente relevante para el análisis, ya que identifica el panel evaluador al que fue asignado cada proyecto, operando como un proxy de la subdisciplina de pertenencia dentro de las Ciencias Sociales.

Sin embargo, dado que GRUPO_DE_EVALUACION contiene categorías de granularidad variable —algunas muy específicas y otras agregadas, con inconsistencias de nomenclatura producto de distintas versiones del sistema de registro— fue necesario construir una variable derivada, GRUPO_ESTUDIO, que agrupa las 101 categorías originales en nueve áreas disciplinares comparables: Antropología y Arqueología; Artes y Arquitectura; Sociología, Trabajo Social y Ciencias de la Comunicación; Geografía y Urbanismo; Psicología; Ciencias Económicas y Administrativas; Ciencias Jurídicas y Políticas; Educación; e Investigación Interdisciplinaria. La clasificación se realizó mediante correspondencia directa entre los valores observados en la base y las categorías de destino, priorizando la coherencia disciplinar por sobre la agrupación administrativa original. Todos los casos fueron clasificados satisfactoriamente, sin observaciones residuales.

Adicionalmente, se incluyen variables geográficas como REGION_EJECUCION y MACROZONA_MINCIENCIA, y variables sobre las características de los beneficiarios e instituciones albergantes como SEXO, TIPO_BENEFICIARIO, NOMBRE_RESPONSABLE e INSTITUCION_PRINCIPAL.

El proceso de depuración aplicado contempló la conversión de variables numéricas relacionadas con montos y años a los tipos de datos apropiados (numeric e integer), la transformación de variables categóricas a factores, y la estandarización de cadenas de texto (incluyendo la corrección de errores tipográficos de origen en los registros oficiales). Finalmente, se aplicó un doble filtrado por disciplina y período temporal, ordenando los registros para facilitar los análisis longitudinales y comparativos posteriores. Es importante destacar que esta base de adjudicaciones, al contener solo proyectos financiados, se utilizó para enriquecer la base de postulaciones mediante un cruce que permitió inyectar información adicional de los proyectos adjudicados a cada postulación correspondiente, facilitando así un análisis integral del proceso competitivo en el programa Fondecyt.

El proceso de depuración consistió en la conversión de variables numéricas a los tipos de datos apropiados, la transformación de variables categóricas a factores para optimizar el análisis, la aplicación de filtros por área disciplinaria y período temporal, y el ordenamiento temporal de los registros. Es importante destacar que la expresión de montos en millones de pesos chilenos facilita la comparación entre proyectos de gran envergadura y refleja la escala significativamente mayor de estos instrumentos de financiamiento comparados con los Fondecyt regulares.

3.2.4 Descripción de la base de postulaciones Proyectos Milenio

La base de datos POSTULACIONES_MILENIO contiene el registro de las propuestas presentadas al Programa Iniciativa Científica Milenio (actualmente bajo la Agencia Nacional de Investigación y Desarrollo, ANID). Si bien la base original abarca el historial de postulaciones a Institutos y Núcleos Milenio desde 1999, para efectos de esta investigación se aplicó un filtro temporal que restringe la muestra al período 2016-2025, garantizando la comparabilidad longitudinal con la base de postulaciones Fondecyt. Asimismo, el universo de análisis se delimitó exclusivamente a los centros de investigación de la disciplina “Ciencias Sociales” (según la clasificación de la OCDE), conservando únicamente aquellas propuestas cuyo proceso de evaluación culminó en condición de “Adjudicado” o “No Adjudicado”.

Entre las variables principales se encuentran el CODIGO_PROYECTO como identificador único, variables de clasificación institucional como SUBDIRECCION, PROGRAMA e INSTRUMENTO (que especifican si se trata de un Instituto o un Núcleo Milenio), y variables temporales como AGNO_FALLO. El resultado del concurso se captura en la variable ESTADO_DE_ADJUDICACION. La base también incluye caracterizaciones geográficas (REGION_EJECUCION y MACROZONA_MINCIENCIA), así como variables sociodemográficas e institucionales vinculadas a la dirección del centro propuesto, tales como SEXO, TIPO_BENEFICIARIO y TIPO_INSTITUCION.

Una limitación relevante de esta fuente refiere a la cobertura temporal de los datos de postulaciones. Si bien la base POSTULACIONES_MILENIO contempla en principio el historial de propuestas presentadas al programa, los registros disponibles para el período de análisis se restringen al año 2017, lo que impide construir series longitudinales de tasas de postulación y adjudicación comparables a las disponibles para Fondecyt. En consecuencia, los datos de postulaciones Milenio se utilizan de manera acotada como referencia ilustrativa para ese año, sin extender su uso a análisis de tendencias temporales.

Para el análisis longitudinal del programa Milenio en el período 2016-2025, el estudio se apoya exclusivamente en la base de proyectos adjudicados, que sí cuenta con cobertura completa del período. Esto implica que, a diferencia de Fondecyt —donde es posible calcular tasas de éxito y caracterizar el universo de postulantes—, el análisis de Milenio se concentra en describir el perfil y la distribución de los centros financiados, sin poder dar cuenta del conjunto de propuestas que no lograron adjudicación en cada concurso.

El proceso de depuración aplicado a esta matriz contempló una primera fase de homologación estructural, donde se renombraron variables clave (eliminando separadores de punto y unificando nomenclaturas) para hacerlas estrictamente consistentes con la estructura de la base Fondecyt. Posteriormente, se eliminaron columnas redundantes, se estandarizaron las cadenas de texto (eliminación de tildes, espacios en blanco y conversión a mayúsculas, incluyendo la limpieza de prefijos regionales), y se transformaron las variables textuales a formato de factor (factor) para optimizar los modelos estadísticos y el cálculo de tasas de adjudicación en investigación asociativa.

3.3 Resumen de las transformaciones aplicadas

El proceso de preparación de datos aplicado sistemáticamente a todas las bases contempló múltiples etapas de transformación y estandarización. En primer lugar, se realizó una estandarización de tipos de datos donde las variables numéricas relacionadas con años, montos y duraciones fueron convertidas a los tipos integer o numeric según correspondiera, mientras que las variables categóricas se transformaron a tipo factor para optimizar tanto el uso de memoria como los análisis estadísticos posteriores. Adicionalmente, se llevó a cabo el renombramiento de variables que contenían caracteres especiales como puntos o espacios, reemplazándolos por guiones bajos para mantener consistencia con las convenciones de nomenclatura en R y evitar problemas de sintaxis.

En segundo lugar, se aplicaron filtros uniformes a todas las bases de datos. El filtro disciplinario seleccionó únicamente proyectos clasificados como “Ciencias Sociales” según la taxonomía OECD, asegurando la homogeneidad del objeto de estudio. El filtro temporal, aplicado especialmente a las bases históricas que contenían datos previos a 2016, restringió el análisis al período 2016-2025, permitiendo mantener consistencia temporal entre todas las bases y concentrarse en la década más reciente de financiamiento científico en Ciencias Sociales.

En tercer lugar, todas las bases fueron ordenadas sistemáticamente por AGNO_FALLO y CODIGO_PROYECTO, facilitando los análisis temporales, longitudinales y la identificación de patrones a lo largo del tiempo. Este ordenamiento resulta fundamental para rastrear trayectorias de investigadores e instituciones, así como para analizar tendencias en el financiamiento.

Finalmente, es importante destacar que todas las bases comparten un conjunto común de variables clave que permiten su comparación y eventual integración. Entre estas variables se encuentran identificadores únicos como CODIGO_PROYECTO y N, variables temporales como AGNO_FALLO y AGNO_CONCURSO (cuando aplica), variables de clasificación institucional como SUBDIRECCION, PROGRAMA e INSTRUMENTO, variables geográficas como REGION_EJECUCION y MACROZONA_MINCIENCIA, variables demográficas e institucionales como SEXO, TIPO_BENEFICIARIO y TIPO_INSTITUCION, y variables disciplinarias como DISCIPLINA_OECD (o AREA_OCDE) y DISCIPLINA_DETALLE. Esta estructura común garantiza la comparabilidad y la posibilidad de realizar análisis integrados entre diferentes instrumentos de financiamiento.
