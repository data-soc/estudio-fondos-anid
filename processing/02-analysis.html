1 Presentacion

Este documento presenta un analisis descriptivo preliminar de las bases procesadas ubicadas en input/data/proc. El objetivo es revisar volumen, composicion, distribuciones clave y una aproximacion inicial a tasas de adjudicacion por anio.

6 Composicion por sexo y region

resumen_categorico <- function(df, var, top_n = 10) {
    if (!var %in% names(df)) return(tibble())

    df %>%
        count(.data[[var]], sort = TRUE, name = "n") %>%
        mutate(prop = n / sum(n)) %>%
        slice_head(n = top_n) %>%
        mutate(
            categoria = as.character(.data[[var]]),
            porcentaje = scales::percent(prop, accuracy = 0.1)
        ) %>%
        select(categoria, n, porcentaje)
}

sexo_post_f <- resumen_categorico(postulaciones, "SEXO", top_n = 5)
sexo_adj_f <- resumen_categorico(adjudicacion, "SEXO", top_n = 5)

region_post_f <- resumen_categorico(postulaciones, "REGION_EJECUCION", top_n = 10)
region_adj_f <- resumen_categorico(adjudicacion, "REGION_EJECUCION", top_n = 10)

resumen_montos_fondecyt <- adjudicacion %>%
    summarise(
        n = n(),
        minimo = min(MONTO_ADJUDICADO, na.rm = TRUE),
        p25 = quantile(MONTO_ADJUDICADO, 0.25, na.rm = TRUE),
        mediana = median(MONTO_ADJUDICADO, na.rm = TRUE),
        promedio = mean(MONTO_ADJUDICADO, na.rm = TRUE),
        p75 = quantile(MONTO_ADJUDICADO, 0.75, na.rm = TRUE),
        maximo = max(MONTO_ADJUDICADO, na.rm = TRUE)

8 Tasas de adjudicacion preliminares

tasa_fondecyt <- postulaciones %>%
    count(AGNO_FALLO, name = "postuladas") %>%
    left_join(adjudicacion %>% count(AGNO_FALLO, name = "adjudicadas"), by = "AGNO_FALLO") %>%
    mutate(
        adjudicadas = replace_na(adjudicadas, 0),
        tasa = adjudicadas / postuladas,
        programa = "Fondecyt"
    )

tasa_milenio <- postulacion_milenio %>%
    count(AGNO_FALLO, name = "postuladas") %>%
    left_join(adjudicacion_milenio %>% count(AGNO_FALLO, name = "adjudicadas"), by = "AGNO_FALLO") %>%
    mutate(
        adjudicadas = replace_na(adjudicadas, 0),
        tasa = adjudicadas / postuladas,
        programa = "Milenio"
    )

tasa_anual <- bind_rows(tasa_fondecyt, tasa_milenio)

tasa_anual %>%
    mutate(tasa = scales::percent(tasa, accuracy = 0.1)) %>%

        caption = "Tasa de adjudicacion anual (aproximacion preliminar)",

9 Comentarios preliminares

  1. Este reporte permite identificar variaciones de volumen por anio, cambios de composicion y diferencias preliminares entre postulacion y adjudicacion.
  2. La tasa de adjudicacion aqui mostrada es descriptiva y depende de la coherencia entre universos anuales de postulaciones y adjudicaciones.
  3. Como siguiente paso, conviene estandarizar definiciones por instrumento y limpiar posibles inconsistencias de CODIGO_PROYECTO para analisis inferenciales.
